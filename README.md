# zarql

graphql playground using zalando public api for getting articles

## Installation
Just run
```
$ yarn
```

## Development
Just run
```
$ npm run dev
```

and enjoy ```localhost:4000/graphql```

## Try it out!
```
http://localhost:4000/graphiql
```
